# poweroff

Windows shutdown scheduler with system tray integration. Allows scheduling automatic shutdown, restart, or logoff at a specified time.

## Features

- Timer-based scheduling with hour/minute/second picker
- System tray icon for background operation
- Supports shutdown, restart, and logoff actions
- Windows privilege escalation for clean shutdown (SE_SHUTDOWN_NAME)

Built with C++ Builder 6.0.
